# Nobu

Nobu is a personal voice AI assistant for memory, planning, style, shopping, and everyday decisions.

ScanFit is one of Nobu's built-in capabilities. It gives honest outfit feedback, smart sizing guidance, look-history support, and styling help without taking over the whole Nobu identity.

Nobu is personal-first, but flexible enough for school, creative work, business, and team projects.

## Development

```bash
npm run dev
```

## Checks

```bash
npm run lint
npm run build
```

## PWA

Nobu is currently focused on the web/PWA experience. The native shells are paused so the main product can move faster.
